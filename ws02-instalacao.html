💿 Instalação Windows Server 2025

Aula 2 — Instale o Windows Server 2025 em uma Máquina Virtual no VirtualBox: da criação da VM ao primeiro login no servidor

💻 Requisitos de Hardware

Antes de instalar, verifique se o hardware (ou a VM) atende aos requisitos mínimos do Windows Server 2025:

Componente Mínimo Recomendado (Lab)
Processador 1,4 GHz 64-bit 2 núcleos (vCPUs)
Memória RAM 512 MB (Server Core) / 2 GB (Desktop) 4 GB
Disco 32 GB 60 GB
Rede 1 adaptador Ethernet 1 adaptador (modo Bridge ou NAT)
Firmware UEFI / TPM 2.0 recomendado UEFI habilitado na VM

✅ Para o laboratório: crie a VM com 4 GB de RAM e 60 GB de disco. Com menos que isso, a instalação fica lenta e a experiência prejudicada.

📥 Obtendo a ISO do Windows Server 2025

A Microsoft disponibiliza uma versão de avaliação gratuita de 180 dias do Windows Server 2025, ideal para uso em laboratório.

🌐 Acesse microsoft.com/evalcenter e busque por Windows Server 2025. Faça o cadastro gratuito e baixe a ISO (aproximadamente 5 GB).

Existem duas opções de download:

💿 ISO 64-bit

  • Arquivo único para gravar em DVD ou montar em VM
  • Inclui Desktop Experience e Server Core
  • Ideal para VirtualBox
  • Tamanho: ~5 GB

📦 VHDX pré-instalado

  • Disco virtual pronto para usar no Hyper-V
  • Não requer instalação
  • Não funciona diretamente no VirtualBox
  • Tamanho: ~8 GB

✅ Baixe a opção ISO 64-bit para usar no VirtualBox.

📦 Criando a VM no VirtualBox

Com a ISO baixada, crie uma nova máquina virtual no VirtualBox seguindo os passos abaixo:

VirtualBox — Nova VM
▶ Nome: WinServer2025
▶ Tipo: Microsoft Windows
▶ Versão: Windows 2022 (64-bit) # ou Windows Server 2025 se disponível
▶ RAM: 4096 MB (4 GB)
▶ Disco: VDI dinâmico — 60 GB
▶ Firmware: UEFI (em Configurações → Sistema)

Após criar a VM, ajuste estas configurações antes de iniciar:

Sistema → Processador 2 CPUs Ativa 2 núcleos para melhor desempenho
Tela → Memória de Vídeo 128 MB Evita tela preta na instalação gráfica
Rede → Adaptador 1 NAT ou Bridge NAT para internet; Bridge para rede local
Armazenamento → Ótico Montar ISO Selecionar a ISO baixada do Eval Center

📥 Instalação Passo a Passo

Inicie a VM. O instalador do Windows Server 2025 será carregado automaticamente pela ISO.

1
Configurações regionais
Selecione Idioma: Português (Brasil), Formato de hora e moeda, e Layout de teclado ABNT2. Clique em Avançar → Instalar agora.
2
Escolha da edição
Selecione Windows Server 2025 Standard Evaluation (Desktop Experience). A opção sem "Desktop Experience" instala o Server Core (sem interface gráfica).
3
Tipo de instalação
Escolha Personalizada: instalar somente o Windows. Na próxima tela, selecione o disco disponível e clique em Avançar. A instalação começará automaticamente.
4
Aguardar e reiniciar
O instalador copia arquivos, instala recursos e reinicia a VM automaticamente (pode reiniciar 2–3 vezes). O processo leva entre 10 e 20 minutos dependendo do hardware.
5
Definir senha do Administrador
Após a instalação, o sistema pede a senha do usuário Administrator. Use uma senha forte (ex: Senha@2025). A senha deve ter letras maiúsculas, minúsculas, números e símbolos.
6
Primeiro login
Pressione Ctrl + Alt + Del (ou Entrada → Inserir Ctrl+Alt+Del no VirtualBox). Digite a senha definida e acesse o Desktop do Windows Server 2025.

⚠️ Atenção: após o primeiro login, o Server Manager abre automaticamente. Não feche — vamos usá-lo na próxima aula para configurar o servidor.

🔧 Instalar Guest Additions (VirtualBox)

As Guest Additions são drivers extras que melhoram a integração entre a VM e o sistema hospedeiro. Instale logo após o primeiro login.

Resolução Tela dinâmica A VM se ajusta ao tamanho da janela
Clipboard Copiar e Colar Compartilha área de transferência com o host
Pastas Shared Folders Acesso a pastas do computador físico
Mouse Integração total Sem precisar pressionar Host para sair da VM
Como instalar as Guest Additions
1. No menu do VirtualBox: Dispositivos → Inserir imagem de CD das Guest Additions
2. Dentro da VM, abra o Explorador de Arquivos → Este Computador
3. Clique duas vezes em VirtualBox Guest Additions (D:)
4. Execute VBoxWindowsAdditions-amd64.exe como Administrador
5. Siga o assistente e reinicie a VM ao final

📸 Snapshot: Ponto de Restauração

Antes de fazer qualquer configuração, crie um Snapshot da VM. É como um ponto de restauração — se algo der errado, você volta ao estado atual em segundos.

Criando um Snapshot no VirtualBox
1. Com a VM ligada ou desligada, clique no menu Máquina → Tirar Instantâneo...
2. Dê um nome descritivo: "Instalação limpa — sem configuração"
3. Clique em OK
✔ Pronto! Para restaurar: Máquina → Restaurar instantâneo

✅ Boas práticas: crie snapshots antes de grandes mudanças: instalação limpa, antes de promover a DC, antes de instalar roles. Mantenha no máximo 3–4 snapshots — cada um ocupa espaço em disco.

📝 Atividade Prática

Instalar o Windows Server 2025 em uma máquina virtual do zero.

1
Baixar a ISO
Acesse o Microsoft Eval Center, faça o cadastro e baixe a ISO do Windows Server 2025 (64-bit). Anote o tamanho do arquivo baixado.
2
Criar e configurar a VM
Crie a VM no VirtualBox com as configurações recomendadas: 4 GB RAM, 60 GB disco VDI dinâmico, 2 vCPUs, UEFI habilitado, 128 MB de vídeo. Anexe a ISO ao drive ótico.
3
Instalar o sistema
Execute a instalação completa: idioma português/ABNT2, edição Standard Desktop Experience, instalação personalizada. Defina a senha do Administrator e faça o primeiro login.
4
Finalizar e documentar
Instale as Guest Additions, crie um snapshot chamado "Instalação limpa" e tire um screenshot do desktop com o Server Manager aberto. Entregue o print como comprovação.
📌 Para refletir: a instalação do Windows Server é muito similar à do Windows desktop, mas com foco em estabilidade. O Server Manager que abre automaticamente é o centro de controle do servidor — é por ele que vamos adicionar todos os papéis nas próximas aulas.

Qual a finalidade das Guest Additions no VirtualBox?

AAumentar a memória RAM disponível para a VM
BInstalar antivírus na máquina virtual
CMelhorar integração entre VM e host: tela dinâmica, clipboard e pastas compartilhadas
DAtivar a licença do Windows Server automaticamente
✓ Guest Additions instala drivers que melhoram a integração: resolução dinâmica, clipboard bidirecional, pastas compartilhadas e integração de mouse.

Para quê serve um Snapshot no VirtualBox?

ASalvar o estado atual da VM para poder restaurar caso algo dê errado
BFazer backup dos arquivos pessoais do usuário
CTirar uma foto da tela da máquina virtual
DCopiar a VM para outro computador
✓ Snapshot salva o estado completo da VM (disco + memória). Se algo der errado após uma configuração, basta restaurar o snapshot e voltar ao ponto seguro.

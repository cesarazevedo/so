🪟 Windows Server 2025: Conceitos

Aula 1 — Conheça o histórico, as edições, os principais papéis e o papel central do Active Directory na infraestrutura corporativa Microsoft

🖥️ O que é Windows Server

O Windows Server é uma família de sistemas operacionais da Microsoft projetada exclusivamente para servidores. Diferente do Windows doméstico (Windows 11), ele é otimizado para:

🏠 Windows Desktop

  • Uso pessoal e profissional
  • Interface gráfica completa
  • Jogos, múdia, produtividade
  • 1 usuário ativo por vez
  • Reinicializações frequentes de updates

🏢 Windows Server

  • Infraestrutura e serviços corporativos
  • Pode rodar sem interface gráfica (Core)
  • Active Directory, DNS, DHCP, File Server
  • Múltiplos usuários simultâneos (RDS)
  • Uptime máximo, updates controlados

💡 Analogia: Windows Desktop é como um carro de passeio — confortável para o dia a dia. Windows Server é como um caminhão de carga — projetado para trabalho pesado, rodando 24h por dia, 7 dias por semana.

📅 Histórico e Versões

A Microsoft lança uma nova versão do Windows Server aproximadamente a cada 2–3 anos. Cada versão traz melhorias de segurança, desempenho e novos recursos.

Versão Ano Destaque Suporte
Windows Server 2012 R2 2013 Hyper-V avançado, Storage Spaces Encerrado 2023
Windows Server 2016 2016 Containers Windows, Nano Server Até 2027
Windows Server 2019 2018 Windows Admin Center, Linux Subsystem Até 2029
Windows Server 2022 2021 Secured-core, TLS 1.3, SMB compactado Até 2031
Windows Server 2025 ⭐ 2024 Hotpatching, SMB over QUIC, AD modernizado Até 2034

✅ Dica: Em ambiente corporativo, sempre use versões com suporte ativo (mainstream + extended). O Windows Server 2025 é a versão mais atual e terá suporte até 2034.

📦 Edições do Windows Server 2025

O Windows Server 2025 está disponível em três edições principais, cada uma para um tipo de ambiente:

Essentials Pequenas empresas Até 25 usuários, sem virtualização ilimitada
Standard Uso geral Até 2 VMs ou containers Hyper-V por licença
Datacenter Nuvem e virtualização VMs ilimitadas, todos os recursos avançados

Além das edições, há duas opções de instalação:

🖥️ Desktop Experience

  • Interface gráfica completa (GUI)
  • Recomendado para aprendizado
  • Mais fácil de administrar visualmente
  • Maior consumo de recursos

💻 Server Core

  • Sem interface gráfica (apenas linha de comando)
  • Menor superfície de ataque
  • Menor consumo de RAM e disco
  • Preferível em produção corporativa

🧩 Papéis e Recursos (Roles & Features)

O Windows Server usa o conceito de Roles (Papéis) e Features (Recursos). Você instala apenas o que precisa, mantendo o servidor enxuto e seguro.

AD DS Active Directory Gerência domínio, usuários e políticas
DNS Servidor DNS Resolução de nomes na rede
DHCP Servidor DHCP Distribui IPs automaticamente
IIS Servidor Web Hospeda sites e aplicações .NET
File Server Servidor de Arquivos Compartilhamentos SMB, cotas e DFS
Hyper-V Virtualização Cria e gerencia máquinas virtuais

💡 Analogia: Pense no Windows Server como um canivete suíço. Você não usa todas as ferramentas ao mesmo tempo — instala apenas o que o ambiente exige. Isso é mais seguro e eficiente.

🏰 O que é Active Directory

O Active Directory Domain Services (AD DS) é o serviço de diretório da Microsoft. Ele centraliza o gerenciamento de identidades (quem pode entrar) e recursos (o que cada um pode acessar) em toda a rede corporativa.

Principais componentes do AD:

🏷️ Domínio (Domain)

Conjunto de objetos (usuários, computadores, grupos) administrados centralmente. Identificado por um nome DNS como escola.local.

🗂️ Unidade Organizacional (OU)

Pasta dentro do domínio para organizar objetos por departamento (ex: TI, RH, Financeiro).

💻 Domain Controller (DC)

Servidor que executa o AD DS e autentica usuários no domínio. Todo domínio precisa de ao menos um DC.

📜 Group Policy (GPO)

Conjunto de políticas aplicadas a usuários e computadores: senha mínima, papel de parede, instalação de software.

✅ Por que o AD é tão importante? Sem ele, cada servidor teria sua própria lista de usuários e senhas. Com o AD, um único login dá acesso a todos os recursos da empresa — isso se chama Single Sign-On (SSO).

⭐ Novidades do Windows Server 2025

O Windows Server 2025 foi lançado em novembro de 2024 com vários recursos modernos focados em segurança e nuvem híbrida:

Hotpatching Atualizações sem reiniciar Aplica patches de segurança sem downtime
SMB over QUIC Compartilhamento seguro Acesso a arquivos pela internet sem VPN
Secured-core Segurança de hardware Proteção baseada em firmware e TPM 2.0
AD Modernizado Active Directory Novos níveis funcionais e segurança LDAP
NVMe/Storage Armazenamento rápido Suporte nativo a NVMe e Storage Spaces Direct
Azure Arc Nuvem híbrida Integração nativa com Azure para gerência centralizada

⚖️ Windows Server vs Linux Server

Na prática corporativa, os dois coexistem. Entender as diferenças ajuda a escolher a ferramenta certa para cada cenário.

Aspecto Windows Server 2025 Linux (Ubuntu Server)
Licença Paga (Microsoft) Gratuita (open source)
Interface GUI completa + PowerShell Linha de comando + GUI opcional
Active Directory Nativo (AD DS) Via Samba 4 ou integração
Servidor Web IIS Apache / Nginx
Compartilhamento SMB nativo Samba / NFS
Virtualizador Hyper-V KVM / VirtualBox
Uso típico Ambientes corporativos Microsoft Servidores web, cloud, DevOps

💡 Realidade do mercado: a maioria das empresas usa os dois. O Windows Server gerencia usuários (AD) e arquivos internos; o Linux roda aplicações web e serviços de nuvem. Saber ambos é um diferencial valioso.

Requisitos de hardware
Consulte os requisitos mínimos do Windows Server 2025 (CPU, RAM, disco). Compare com o Ubuntu Server 24.04. Qual dos dois exige menos recursos? Registre sua conclusão.

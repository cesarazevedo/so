Transforme comandos repetitivos em scripts poderosos! Crie programas que conversam com usuários, processam dados e automatizam tarefas. Do básico ao primeiro projeto real: um sistema de busca de alunos.

ExitCode Variáveis Interação Bash

IF/ELSE Test Operadores Lógica

🔄 Shell Script - Estrutura CASE

Múltiplas escolhas de forma elegante! Domine a estrutura CASE para criar menus interativos, sistemas de opções e scripts que lidam com várias possibilidades. Quando IF não é suficiente!

CASE Padrões Menus Wildcards

🔁 Shell Script - Loops FOR

Repetindo tarefas de forma inteligente! Domine os loops FOR para automatizar operações, processar arquivos em lote e criar scripts que trabalham com listas. O poder da automação nas suas mãos!

FOR Sequências Arquivos Wildcards

🔄 Shell Script - Loops WHILE

WHILE Condições Interativo Comparadores

14

🌐 Configuração de endereço IP estático

Configure endereços IP fixos no Linux! Aprenda a usar Netplan para garantir IPs permanentes em servidores e facilitar o acesso remoto. Essencial para ambientes profissionais!.

ifconfig netplan DHCP servidor
✅ Disponível 🌐
15

🌐 Servidor DHCP

Distribua IPs automaticamente! Configure um servidor DHCP com ISC-DHCP para gerenciar centenas de dispositivos sem esforço. Liberte-se da configuração manual!

DHCP Server ISC-DHCP Lease Time Subnet
✅ Disponível 🌐
16

📂 Servidor de Arquivos SAMBA

Conecte mundos diferentes! Domine a configuração do SAMBA para compartilhar arquivos entre Windows e Linux de forma transparente. Crie uma ponte perfeita entre sistemas operacionais distintos e facilite o trabalho colaborativo!

SAMBA SMB/CIFS Windows/Linux smb.conf
✅ Disponível 📂
17

🌍 Servidor Web Apache

Hospede sites profissionais! Configure o Apache, servidor web usado por 30% da internet. Aprenda Virtual Hosts para rodar múltiplos sites em um só servidor!

Apache2 Virtual Hosts HTTP DocumentRoot
✅ Disponível 🌍
18

🌐 Servidor DNS - BIND9

Transforme nomes em endereços IP! Configure o BIND9 para criar seu próprio servidor DNS. Entenda como www.google.com vira 142.250.185.78 na prática!

BIND9 DNS Resolução de Nomes Zonas DNS
✅ Disponível 🌐
19

🔐 Servidor Proxy - SQUID

Controle e monitore o acesso à internet! Configure o SQUID para bloquear sites, acelerar navegação com cache e criar regras personalizadas com ACLs. Segurança e performance!

SQUID Proxy Server ACLs Bloqueio de Sites Cache
✅ Disponível 🔐
20

🖥️ Webmin - Interface de Gerenciamento

Gerencie Linux visualmente! Instale o Webmin e controle seu servidor através de uma interface web moderna. Usuários, serviços e logs sem precisar decorar comandos!

Webmin Interface Web Gerenciamento Painel de Controle Administração
✅ Disponível 🖥️
21

🚀 PROJETO FINAL - Infraestrutura Completa

Integre TUDO que você aprendeu! Crie uma infraestrutura de rede corporativa completa unindo DHCP, SAMBA, BIND9, Apache, SQUID e Webmin em um único ambiente funcional. Mostre seu domínio total dos serviços de rede!

DHCP SAMBA BIND9 Apache SQUID Webmin
